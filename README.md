# Emotion Detector

This project is a web-based Emotion Detection application developed using Python and Flask.

## Features

- Detects emotions from user text.
- Displays the dominant emotion.
- Handles empty input gracefully.
- Includes unit tests.
- Flask web interface.
- Modular Python package.

## Technologies Used

- Python
- Flask
- Git
- Pylint
